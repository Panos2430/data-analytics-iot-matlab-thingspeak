# Data Analytics in the IoT using MATLAB & ThingSpeak

IoT data collection, analysis and visualization using MATLAB and ThingSpeak platform

Αυτό το project περιλαμβάνει την πτυχιακή μου εργασία σχετικά με την ανάλυση δεδομένων από IoT συσκευές χρησιμοποιώντας MATLAB και ThingSpeak.

## **Περιεχόμενα**

- **Docs/**  
  Περιέχει την πτυχιακή εργασία σε PDF.
  
- **ThingSpeak**  
  Για να δείτε τα δεδομένα που συλλέχθηκαν και την ανάλυση σε πραγματικό χρόνο:  
  [ThingSpeak Channel](https://thingspeak.com/channels/XXXXXX)
  
## **Οδηγίες χρήσης**

1. Κατεβάστε το PDF από τον φάκελο `Docs/` για να δείτε όλη την εργασία.  
2. Επισκεφθείτε το ThingSpeak link για να δείτε τα δεδομένα που συλλέχθηκαν και τις αναλύσεις.  

**Σημείωση:** Ο κώδικας MATLAB και τα scripts του ThingSpeak δεν περιλαμβάνονται δημόσια για προστασία της πνευματικής μου ιδιοκτησίας.
